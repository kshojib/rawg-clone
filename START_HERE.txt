You're seeing an error because the RAWG API key isn't configured yet.
Don't worry - it's super easy and FREE! Follow these steps:

┌──────────────────────────────────────────────────────────────┐
│ STEP 1: Get Your Free API Key (30 seconds)                  │
└──────────────────────────────────────────────────────────────┘

1. Open this link: https://rawg.io/apidocs
2. Click "Get API Key"  
3. Sign up (takes 30 seconds - no credit card needed!)
4. Copy your API key from the dashboard

┌──────────────────────────────────────────────────────────────┐
│ STEP 2: Add Your API Key                                    │
└──────────────────────────────────────────────────────────────┘

1. Open the file: .env.local (in this folder)
2. Find this line:
   NEXT_PUBLIC_RAWG_API_KEY=your_api_key_here

3. Replace "your_api_key_here" with your actual key:
   NEXT_PUBLIC_RAWG_API_KEY=abc123yourkeyhere

4. Save the file

┌──────────────────────────────────────────────────────────────┐
│ STEP 3: Refresh the Page                                    │
└──────────────────────────────────────────────────────────────┘

Go back to your browser and refresh the page (F5 or Ctrl+R)

That's it! You should now see games loading! 🎉
